# Thom's COokie CUtter for Python (CoCuP)

## Description
A 'cookie cutter' to create a blank template for a python package in the target directory.

`CoCuP` will create a skeleton Python package with the following architechture:

```
my_package/
├── setup.py
├── README.md
├── my_package/
│   ├── __init__.py
│   ├── __main__.py
│   ├── main.py
│   ├── parser/
│   │   ├── __init__.py
│   │   └── parser.py
│   ├── utils/
│   │   ├── __init__.py
│   │   ├── classes.py
│   │   ├── errors.py
│   │   └── io_utils.py
├── example_data/
│   ├── example_in/
│   └── example_out/
└── tests/
    ├── __init__.py
    ├── test_utils.py
    └── test_main.py
```
